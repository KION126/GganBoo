# GganBoo

### 공유 ToDoList 모바일 애플리케이션
* Android Studio를 사용한 자신의 TodoList를 공유할 수 있는 모바일 애플리케이션
* FireBase를 활용하여 회원관리 및 이메일 인증, 팔로우 구현
* 팔로우한 다른 유저의 TodoList 캘린더를 확인
